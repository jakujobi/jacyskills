# jacyskills

Public global skills by `jakujobi`.

This repo is the public home for the `jacyskills` skill set. I may refer to this set as `JAS`, but the actual repo/package name is `jacyskills`.

## Included Skills

- `engineering-session-journaler`: preserves durable engineering work memory across planning, investigation, implementation, debugging, validation, ops, and hardware work.
- `engineering-clarifier`: researches first, then asks grouped clarification rounds with decision-focused options and recommendations.

## Install

Install specific skills globally:

```bash
npx skills add jakujobi/jacyskills --skill engineering-session-journaler engineering-clarifier -g -y
```

Install all skills from this repo globally:

```bash
npx skills add jakujobi/jacyskills -g -y
```

List globally installed skills:

```bash
npx skills list -g
```
